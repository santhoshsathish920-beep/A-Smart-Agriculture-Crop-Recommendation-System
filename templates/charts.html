Rainfall and Humidity are the strongest predictors, followed closely by Potassium (K). The model assigns weight to each feature based on how much it reduces prediction error.

🎯 Confusion Matrix Model Accuracy

The confusion matrix shows predicted vs. actual crops on the test set. A perfect model would show all values along the diagonal. High diagonal values indicate accurate predictions per class.
